# Yii2+
Yii2 + Bootstrap搭建的后台管理系统，集成用户中心模块和RBAC权限管理模块,前端页面采用H+框架，集成Yii2 admin。

#安装
使用Git clone到本地后，新建数据库，将yii2advanced2.sql文件导入数据库，修改common/config/main-local.php 配置文件数据库信息即可
